1""" 

2A directive for including a Matplotlib plot in a Sphinx document 

3================================================================ 

4 

5This is a Sphinx extension providing a reStructuredText directive 

6``.. plot::`` for including a plot in a Sphinx document. 

7 

8In HTML output, ``.. plot::`` will include a .png file with a link 

9to a high-res .png and .pdf. In LaTeX output, it will include a .pdf. 

10 

11The plot content may be defined in one of three ways: 

12 

131. **A path to a source file** as the argument to the directive:: 

14 

15 .. plot:: path/to/plot.py 

16 

17 When a path to a source file is given, the content of the 

18 directive may optionally contain a caption for the plot:: 

19 

20 .. plot:: path/to/plot.py 

21 

22 The plot caption. 

23 

24 Additionally, one may specify the name of a function to call (with 

25 no arguments) immediately after importing the module:: 

26 

27 .. plot:: path/to/plot.py plot_function1 

28 

292. Included as **inline content** to the directive:: 

30 

31 .. plot:: 

32 

33 import matplotlib.pyplot as plt 

34 plt.plot([1, 2, 3], [4, 5, 6]) 

35 plt.title("A plotting exammple") 

36 

373. Using **doctest** syntax:: 

38 

39 .. plot:: 

40 

41 A plotting example: 

42 >>> import matplotlib.pyplot as plt 

43 >>> plt.plot([1, 2, 3], [4, 5, 6]) 

44 

45Options 

46------- 

47 

48The ``.. plot::`` directive supports the following options: 

49 

50 ``:format:`` : {'python', 'doctest'} 

51 The format of the input. If unset, the format is auto-detected. 

52 

53 ``:include-source:`` : bool 

54 Whether to display the source code. The default can be changed using 

55 the ``plot_include_source`` variable in :file:`conf.py` (which itself 

56 defaults to False). 

57 

58 ``:show-source-link:`` : bool 

59 Whether to show a link to the source in HTML. The default can be 

60 changed using the ``plot_html_show_source_link`` variable in 

61 :file:`conf.py` (which itself defaults to True). 

62 

63 ``:context:`` : bool or str 

64 If provided, the code will be run in the context of all previous plot 

65 directives for which the ``:context:`` option was specified. This only 

66 applies to inline code plot directives, not those run from files. If 

67 the ``:context: reset`` option is specified, the context is reset 

68 for this and future plots, and previous figures are closed prior to 

69 running the code. ``:context: close-figs`` keeps the context but closes 

70 previous figures before running the code. 

71 

72 ``:nofigs:`` : bool 

73 If specified, the code block will be run, but no figures will be 

74 inserted. This is usually useful with the ``:context:`` option. 

75 

76 ``:caption:`` : str 

77 If specified, the option's argument will be used as a caption for the 

78 figure. This overwrites the caption given in the content, when the plot 

79 is generated from a file. 

80 

81Additionally, this directive supports all the options of the `image directive 

82<https://docutils.sourceforge.io/docs/ref/rst/directives.html#image>`_, 

83except for ``:target:`` (since plot will add its own target). These include 

84``:alt:``, ``:height:``, ``:width:``, ``:scale:``, ``:align:`` and ``:class:``. 

85 

86Configuration options 

87--------------------- 

88 

89The plot directive has the following configuration options: 

90 

91 plot_include_source 

92 Default value for the include-source option (default: False). 

93 

94 plot_html_show_source_link 

95 Whether to show a link to the source in HTML (default: True). 

96 

97 plot_pre_code 

98 Code that should be executed before each plot. If None (the default), 

99 it will default to a string containing:: 

100 

101 import numpy as np 

102 from matplotlib import pyplot as plt 

103 

104 plot_basedir 

105 Base directory, to which ``plot::`` file names are relative to. 

106 If None or empty (the default), file names are relative to the 

107 directory where the file containing the directive is. 

108 

109 plot_formats 

110 File formats to generate (default: ['png', 'hires.png', 'pdf']). 

111 List of tuples or strings:: 

112 

113 [(suffix, dpi), suffix, ...] 

114 

115 that determine the file format and the DPI. For entries whose 

116 DPI was omitted, sensible defaults are chosen. When passing from 

117 the command line through sphinx_build the list should be passed as 

118 suffix:dpi,suffix:dpi, ... 

119 

120 plot_html_show_formats 

121 Whether to show links to the files in HTML (default: True). 

122 

123 plot_rcparams 

124 A dictionary containing any non-standard rcParams that should 

125 be applied before each plot (default: {}). 

126 

127 plot_apply_rcparams 

128 By default, rcParams are applied when ``:context:`` option is not used 

129 in a plot directive. If set, this configuration option overrides this 

130 behavior and applies rcParams before each plot. 

131 

132 plot_working_directory 

133 By default, the working directory will be changed to the directory of 

134 the example, so the code can get at its data files, if any. Also its 

135 path will be added to `sys.path` so it can import any helper modules 

136 sitting beside it. This configuration option can be used to specify 

137 a central directory (also added to `sys.path`) where data files and 

138 helper modules for all code are located. 

139 

140 plot_template 

141 Provide a customized template for preparing restructured text.

176def _option_boolean(arg): 

177 if not arg or not arg.strip(): 

178 # no argument given, assume used as a flag 

179 return True 

180 elif arg.strip().lower() in ('no', '0', 'false'): 

181 return False 

182 elif arg.strip().lower() in ('yes', '1', 'true'): 

183 return True 

184 else: 

185 raise ValueError(f'{arg!r} unknown boolean')

189 if arg in [None, 'reset', 'close-figs']: 

190 return arg 

191 raise ValueError("Argument should be None or 'reset' or 'close-figs'")

198def mark_plot_labels(app, document): 

199 """ 

200 To make plots referenceable, we need to move the reference from the 

201 "htmlonly" (or "latexonly") node to the actual figure node itself. 

202 """ 

203 for name, explicit in document.nametypes.items(): 

204 if not explicit: 

205 continue 

206 labelid = document.nameids[name] 

207 if labelid is None: 

208 continue 

209 node = document.ids[labelid] 

210 if node.tagname in ('html_only', 'latex_only'): 

211 for n in node: 

212 if n.tagname == 'figure': 

213 sectname = name 

214 for c in n: 

215 if c.tagname == 'caption': 

216 sectname = c.astext() 

217 break 

218 

219 node['ids'].remove(labelid) 

220 node['names'].remove(name) 

221 n['ids'].append(labelid) 

222 n['names'].append(name) 

223 document.settings.env.labels[name] = \ 

224 document.settings.env.docname, labelid, sectname 

225 break

228class PlotDirective(Directive): 

229 """The ``.. plot::`` directive, as documented in the module's docstring.""" 

230 

231 has_content = True 

232 required_arguments = 0 

233 optional_arguments = 2 

234 final_argument_whitespace = False 

235 option_spec = { 

236 'alt': directives.unchanged, 

237 'height': directives.length_or_unitless, 

238 'width': directives.length_or_percentage_or_unitless, 

239 'scale': directives.nonnegative_int, 

240 'align': Image.align, 

241 'class': directives.class_option, 

242 'include-source': _option_boolean, 

243 'show-source-link': _option_boolean, 

244 'format': _option_format, 

245 'context': _option_context, 

246 'nofigs': directives.flag, 

247 'caption': directives.unchanged, 

248 }

250 def run(self): 

251 """Run the plot directive.""" 

252 try: 

253 return run(self.arguments, self.content, self.options, 

254 self.state_machine, self.state, self.lineno) 

255 except Exception as e: 

256 raise self.error(str(e))

296def contains_doctest(text): 

297 try: 

298 # check if it's valid Python as-is 

299 compile(text, '<string>', 'exec') 

300 return False 

301 except SyntaxError: 

302 pass 

303 r = re.compile(r'^\s*>>>', re.M) 

304 m = r.search(text) 

305 return bool(m)

308def _split_code_at_show(text, function_name): 

309 """Split code at plt.show().""" 

310 

311 is_doctest = contains_doctest(text) 

312 if function_name is None: 

313 parts = [] 

314 part = [] 

315 for line in text.split("\n"): 

316 if ((not is_doctest and line.startswith('plt.show(')) or 

317 (is_doctest and line.strip() == '>>> plt.show()')): 

318 part.append(line) 

319 parts.append("\n".join(part)) 

320 part = [] 

321 else: 

322 part.append(line) 

323 if "\n".join(part).strip(): 

324 parts.append("\n".join(part)) 

325 else: 

326 parts = [text] 

327 return is_doctest, parts

413def out_of_date(original, derived, includes=None): 

414 """ 

415 Return whether *derived* is out-of-date relative to *original* or any of 

416 the RST files included in it using the RST include directive (*includes*). 

417 *derived* and *original* are full paths, and *includes* is optionally a 

418 list of full paths which may have been included in the *original*. 

419 """ 

420 if not os.path.exists(derived): 

421 return True 

422 

423 if includes is None: 

424 includes = [] 

425 files_to_check = [original, *includes] 

426 

427 def out_of_date_one(original, derived_mtime): 

428 return (os.path.exists(original) and 

429 derived_mtime < os.stat(original).st_mtime) 

430 

431 derived_mtime = os.stat(derived).st_mtime 

432 return any(out_of_date_one(f, derived_mtime) for f in files_to_check)

439def _run_code(code, code_path, ns=None, function_name=None): 

440 """ 

441 Import a Python module from a path, and run the function given by 

442 name, if function_name is not None. 

443 """ 

444 

445 # Change the working directory to the directory of the example, so 

446 # it can get at its data files, if any. Add its path to sys.path 

447 # so it can import any helper modules sitting beside it. 

448 pwd = os.getcwd() 

449 if setup.config.plot_working_directory is not None: 

450 try: 

451 os.chdir(setup.config.plot_working_directory) 

452 except OSError as err: 

453 raise OSError(f'{err}\n`plot_working_directory` option in ' 

454 f'Sphinx configuration file must be a valid ' 

455 f'directory path') from err 

456 except TypeError as err: 

457 raise TypeError(f'{err}\n`plot_working_directory` option in ' 

458 f'Sphinx configuration file must be a string or ' 

459 f'None') from err 

460 elif code_path is not None: 

461 dirname = os.path.abspath(os.path.dirname(code_path)) 

462 os.chdir(dirname) 

463 

464 with cbook._setattr_cm( 

465 sys, argv=[code_path], path=[os.getcwd(), *sys.path]), \ 

466 contextlib.redirect_stdout(StringIO()): 

467 try: 

468 if ns is None: 

469 ns = {} 

470 if not ns: 

471 if setup.config.plot_pre_code is None: 

472 exec('import numpy as np\n' 

473 'from matplotlib import pyplot as plt\n', ns) 

474 else: 

475 exec(str(setup.config.plot_pre_code), ns) 

476 if "__main__" in code: 

477 ns['__name__'] = '__main__' 

478 

479 # Patch out non-interactive show() to avoid triggering a warning. 

480 with cbook._setattr_cm(FigureManagerBase, show=lambda self: None): 

481 exec(code, ns) 

482 if function_name is not None: 

483 exec(function_name + "()", ns) 

484 

485 except (Exception, SystemExit) as err: 

486 raise PlotError(traceback.format_exc()) from err 

487 finally: 

488 os.chdir(pwd) 

489 return ns

499def get_plot_formats(config): 

500 default_dpi = {'png': 80, 'hires.png': 200, 'pdf': 200} 

501 formats = [] 

502 plot_formats = config.plot_formats 

503 for fmt in plot_formats: 

504 if isinstance(fmt, str): 

505 if ':' in fmt: 

506 suffix, dpi = fmt.split(':') 

507 formats.append((str(suffix), int(dpi))) 

508 else: 

509 formats.append((fmt, default_dpi.get(fmt, 80))) 

510 elif isinstance(fmt, (tuple, list)) and len(fmt) == 2: 

511 formats.append((str(fmt[0]), int(fmt[1]))) 

512 else: 

513 raise PlotError('invalid image format "%r" in plot_formats' % fmt) 

514 return formats

517def render_figures(code, code_path, output_dir, output_base, context, 

518 function_name, config, context_reset=False, 

519 close_figs=False, 

520 code_includes=None): 

521 """ 

522 Run a pyplot script and save the images in *output_dir*. 

523 

524 Save the images under *output_dir* with file names derived from 

525 *output_base* 

526 """ 

527 if function_name is not None: 

528 output_base = f'{output_base}_{function_name}' 

529 formats = get_plot_formats(config) 

530 

531 # Try to determine if all images already exist 

532 

533 is_doctest, code_pieces = _split_code_at_show(code, function_name) 

534 

535 # Look for single-figure output files first 

536 img = ImageFile(output_base, output_dir) 

537 for format, dpi in formats: 

538 if context or out_of_date(code_path, img.filename(format), 

539 includes=code_includes): 

540 all_exists = False 

541 break 

542 img.formats.append(format) 

543 else: 

544 all_exists = True 

545 

546 if all_exists: 

547 return [(code, [img])] 

548 

549 # Then look for multi-figure output files 

550 results = [] 

551 for i, code_piece in enumerate(code_pieces): 

552 images = [] 

553 for j in itertools.count(): 

554 if len(code_pieces) > 1: 

555 img = ImageFile('%s_%02d_%02d' % (output_base, i, j), 

556 output_dir) 

557 else: 

558 img = ImageFile('%s_%02d' % (output_base, j), output_dir) 

559 for fmt, dpi in formats: 

560 if context or out_of_date(code_path, img.filename(fmt), 

561 includes=code_includes): 

562 all_exists = False 

563 break 

564 img.formats.append(fmt) 

565 

566 # assume that if we have one, we have them all 

567 if not all_exists: 

568 all_exists = (j > 0) 

569 break 

570 images.append(img) 

571 if not all_exists: 

572 break 

573 results.append((code_piece, images)) 

574 else: 

575 all_exists = True 

576 

577 if all_exists: 

578 return results 

579 

580 # We didn't find the files, so build them 

581 

582 results = [] 

583 ns = plot_context if context else {} 

584 

585 if context_reset: 

586 clear_state(config.plot_rcparams) 

587 plot_context.clear() 

588 

589 close_figs = not context or close_figs 

590 

591 for i, code_piece in enumerate(code_pieces): 

592 

593 if not context or config.plot_apply_rcparams: 

594 clear_state(config.plot_rcparams, close_figs) 

595 elif close_figs: 

596 plt.close('all') 

597 

598 _run_code(doctest.script_from_examples(code_piece) if is_doctest 

599 else code_piece, 

600 code_path, ns, function_name) 

601 

602 images = [] 

603 fig_managers = _pylab_helpers.Gcf.get_all_fig_managers() 

604 for j, figman in enumerate(fig_managers): 

605 if len(fig_managers) == 1 and len(code_pieces) == 1: 

606 img = ImageFile(output_base, output_dir) 

607 elif len(code_pieces) == 1: 

608 img = ImageFile("%s_%02d" % (output_base, j), output_dir) 

609 else: 

610 img = ImageFile("%s_%02d_%02d" % (output_base, i, j), 

611 output_dir) 

612 images.append(img) 

613 for fmt, dpi in formats: 

614 try: 

615 figman.canvas.figure.savefig(img.filename(fmt), dpi=dpi) 

616 except Exception as err: 

617 raise PlotError(traceback.format_exc()) from err 

618 img.formats.append(fmt) 

619 

620 results.append((code_piece, images)) 

621 

622 if not context or config.plot_apply_rcparams: 

623 clear_state(config.plot_rcparams, close=not context) 

624 

625 return results

628def run(arguments, content, options, state_machine, state, lineno): 

629 document = state_machine.document 

630 config = document.settings.env.config 

631 nofigs = 'nofigs' in options 

632 

633 formats = get_plot_formats(config) 

634 default_fmt = formats[0][0] 

635 

636 options.setdefault('include-source', config.plot_include_source) 

637 options.setdefault('show-source-link', config.plot_html_show_source_link) 

638 if 'class' in options: 

639 # classes are parsed into a list of string, and output by simply 

640 # printing the list, abusing the fact that RST guarantees to strip 

641 # non-conforming characters 

642 options['class'] = ['plot-directive'] + options['class'] 

643 else: 

644 options.setdefault('class', ['plot-directive']) 

645 keep_context = 'context' in options 

646 context_opt = None if not keep_context else options['context'] 

647 

648 rst_file = document.attributes['source'] 

649 rst_dir = os.path.dirname(rst_file) 

650 

651 if len(arguments): 

652 if not config.plot_basedir: 

653 source_file_name = os.path.join(setup.app.builder.srcdir, 

654 directives.uri(arguments[0])) 

655 else: 

656 source_file_name = os.path.join(setup.confdir, config.plot_basedir, 

657 directives.uri(arguments[0])) 

658 

659 # If there is content, it will be passed as a caption. 

660 caption = '\n'.join(content) 

661 

662 # Enforce unambiguous use of captions. 

663 if "caption" in options: 

664 if caption: 

665 raise ValueError( 

666 'Caption specified in both content and options.' 

667 ' Please remove ambiguity.' 

668 ) 

669 # Use caption option 

670 caption = options["caption"] 

671 

672 # If the optional function name is provided, use it 

673 if len(arguments) == 2: 

674 function_name = arguments[1] 

675 else: 

676 function_name = None 

677 

678 code = Path(source_file_name).read_text(encoding='utf-8') 

679 output_base = os.path.basename(source_file_name) 

680 else: 

681 source_file_name = rst_file 

682 code = textwrap.dedent("\n".join(map(str, content))) 

683 counter = document.attributes.get('_plot_counter', 0) + 1 

684 document.attributes['_plot_counter'] = counter 

685 base, ext = os.path.splitext(os.path.basename(source_file_name)) 

686 output_base = '%s-%d.py' % (base, counter) 

687 function_name = None 

688 caption = options.get('caption', '') 

689 

690 base, source_ext = os.path.splitext(output_base) 

691 if source_ext in ('.py', '.rst', '.txt'): 

692 output_base = base 

693 else: 

694 source_ext = '' 

695 

696 # ensure that LaTeX includegraphics doesn't choke in foo.bar.pdf filenames 

697 output_base = output_base.replace('.', '-') 

698 

699 # is it in doctest format? 

700 is_doctest = contains_doctest(code) 

701 if 'format' in options: 

702 if options['format'] == 'python': 

703 is_doctest = False 

704 else: 

705 is_doctest = True 

706 

707 # determine output directory name fragment 

708 source_rel_name = relpath(source_file_name, setup.confdir) 

709 source_rel_dir = os.path.dirname(source_rel_name).lstrip(os.path.sep) 

710 

711 # build_dir: where to place output files (temporarily) 

712 build_dir = os.path.join(os.path.dirname(setup.app.doctreedir), 

713 'plot_directive', 

714 source_rel_dir) 

715 # get rid of .. in paths, also changes pathsep 

716 # see note in Python docs for warning about symbolic links on Windows. 

717 # need to compare source and dest paths at end 

718 build_dir = os.path.normpath(build_dir) 

719 os.makedirs(build_dir, exist_ok=True) 

720 

721 # how to link to files from the RST file 

722 try: 

723 build_dir_link = relpath(build_dir, rst_dir).replace(os.path.sep, '/') 

724 except ValueError: 

725 # on Windows, relpath raises ValueError when path and start are on 

726 # different mounts/drives 

727 build_dir_link = build_dir 

728 

729 # get list of included rst files so that the output is updated when any 

730 # plots in the included files change. These attributes are modified by the 

731 # include directive (see the docutils.parsers.rst.directives.misc module). 

732 try: 

733 source_file_includes = [os.path.join(os.getcwd(), t[0]) 

734 for t in state.document.include_log] 

735 except AttributeError: 

736 # the document.include_log attribute only exists in docutils >=0.17, 

737 # before that we need to inspect the state machine 

738 possible_sources = {os.path.join(setup.confdir, t[0]) 

739 for t in state_machine.input_lines.items} 

740 source_file_includes = [f for f in possible_sources 

741 if os.path.isfile(f)] 

742 # remove the source file itself from the includes 

743 try: 

744 source_file_includes.remove(source_file_name) 

745 except ValueError: 

746 pass 

747 

748 # save script (if necessary) 

749 if options['show-source-link']: 

750 Path(build_dir, output_base + source_ext).write_text( 

751 doctest.script_from_examples(code) 

752 if source_file_name == rst_file and is_doctest 

753 else code, 

754 encoding='utf-8') 

755 

756 # make figures 

757 try: 

758 results = render_figures(code=code, 

759 code_path=source_file_name, 

760 output_dir=build_dir, 

761 output_base=output_base, 

762 context=keep_context, 

763 function_name=function_name, 

764 config=config, 

765 context_reset=context_opt == 'reset', 

766 close_figs=context_opt == 'close-figs', 

767 code_includes=source_file_includes) 

768 errors = [] 

769 except PlotError as err: 

770 reporter = state.memo.reporter 

771 sm = reporter.system_message( 

772 2, "Exception occurred in plotting {}\n from {}:\n{}".format( 

773 output_base, source_file_name, err), 

774 line=lineno) 

775 results = [(code, [])] 

776 errors = [sm] 

777 

778 # Properly indent the caption 

779 caption = '\n' + '\n'.join(' ' + line.strip() 

780 for line in caption.split('\n')) 

781 

782 # generate output restructuredtext 

783 total_lines = [] 

784 for j, (code_piece, images) in enumerate(results): 

785 if options['include-source']: 

786 if is_doctest: 

787 lines = ['', *code_piece.splitlines()] 

788 else: 

789 lines = ['.. code-block:: python', '', 

790 *textwrap.indent(code_piece, ' ').splitlines()] 

791 source_code = "\n".join(lines) 

792 else: 

793 source_code = "" 

794 

795 if nofigs: 

796 images = [] 

797 

798 opts = [ 

799 f':{key}: {val}' for key, val in options.items() 

800 if key in ('alt', 'height', 'width', 'scale', 'align', 'class')] 

801 

802 # Not-None src_name signals the need for a source download in the 

803 # generated html 

804 if j == 0 and options['show-source-link']: 

805 src_name = output_base + source_ext 

806 else: 

807 src_name = None 

808 

809 result = jinja2.Template(config.plot_template or TEMPLATE).render( 

810 default_fmt=default_fmt, 

811 build_dir=build_dir_link, 

812 src_name=src_name, 

813 multi_image=len(images) > 1, 

814 options=opts, 

815 images=images, 

816 source_code=source_code, 

817 html_show_formats=config.plot_html_show_formats and len(images), 

818 caption=caption) 

819 

820 total_lines.extend(result.split("\n")) 

821 total_lines.extend("\n") 

822 

823 if total_lines: 

824 state_machine.insert_input(total_lines, source=source_file_name) 

825 

826 return errors
